# Support and Review Channels

SkyPilot H743 is a review-stage open FPV flight-controller hardware project.

## Best Places to Start

- Contribution hub: https://19379353560.github.io/contribute.html
- Project page: https://19379353560.github.io/skypilot.html
- Hardware review request: https://github.com/19379353560/skypilot/issues/1
- Discussion: https://github.com/19379353560/skypilot/discussions/2
- Good first issue: https://github.com/19379353560/skypilot/issues/3

## Useful Support Topics

- PCB layout, power, grounding, and connector review.
- ICM42688P IMU, DRDY wiring, and vibration-isolation review.
- Safe bring-up notes before powering or flying a prototype.
- INAV target configuration notes for STM32H743-based boards.

For safety-sensitive hardware notes, include photos, board revision, test setup,
power source, current limit, firmware version, and what changed between tests.
