# Roadmap

## Review Stage

- Collect PCB layout and IMU/DRDY wiring feedback.
- Compare grounding, power filtering, and vibration isolation with similar
  STM32H743 flight-controller designs.
- Track review notes in https://github.com/19379353560/skypilot/issues/1.

## Prototype Stage

- Validate firmware target behavior on bench.
- Record sensor health, IMU timing, and basic INAV configuration notes.
- Add photos, wiring notes, and assembly caveats.

## Flight-Test Stage

- Collect short hover and conservative flight-test notes.
- Capture Blackbox logs where possible.
- Update README and project status with confirmed hardware behavior.
